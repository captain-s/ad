# AD Maven 项目文档

本文档旨在帮助开发者快速了解和使用 AD Maven 项目。

## 文档结构

- [项目简介](introduction.md) - 项目概述、技术栈和主要功能
- [快速开始](quick-start.md) - 环境搭建和项目运行
- [路由配置](routing.md) - 路由的配置方式和使用方法
- [权限管理](permission.md) - 权限系统的实现和使用
- [组件库](components.md) - 内置组件的使用指南
- [布局系统](layout.md) - 布局结构和配置
- [主题配置](theme.md) - 主题定制和样式管理
- [API 调用](api.md) - API 服务的使用方式
- [国际化](i18n.md) - 多语言支持配置
- [开发流程](development.md) - 开发规范和构建流程

## 技术栈

- React 19.1.0
- TypeScript
- Vite
- Ant Design 6.3.5
- React Router 7.0.2
- Tailwind CSS 4.1.3
- Zustand (状态管理)
- TanStack Query (数据请求)

## 项目特点

- 现代化的技术栈和架构
- 完整的权限管理系统
- 灵活的路由配置
- 丰富的内置组件
- 支持多语言国际化
- 响应式设计，适配不同设备
- 良好的代码组织和可维护性
