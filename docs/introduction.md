# 项目简介

AD Maven 是一个现代化的管理系统模板，基于 React 19 和 Ant Design 6 构建，提供了完整的管理系统功能和组件。

## 项目特点

- **现代化技术栈**：使用 React 19、TypeScript、Vite 等现代前端技术
- **完整的权限管理**：支持基于角色和权限的访问控制
- **灵活的路由系统**：支持前端和后端路由模式
- **丰富的内置组件**：提供多种常用组件，如导航、表单、图表等
- **响应式设计**：适配不同屏幕尺寸
- **多语言支持**：内置国际化功能
- **主题定制**：支持明暗主题切换和自定义主题

## 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| React | 19.1.0 | 前端框架 |
| TypeScript | 5.6.3 | 类型系统 |
| Vite | 6.2.0 | 构建工具 |
| Ant Design | 6.3.5 | UI 组件库 |
| React Router | 7.0.2 | 路由管理 |
| Tailwind CSS | 4.1.3 | 样式框架 |
| Zustand | 4.5.5 | 状态管理 |
| TanStack Query | 5.60.2 | 数据请求和缓存 |
| i18next | 23.16.5 | 国际化 |

## 项目结构

```
src/
├── _mock/            # 模拟数据
├── api/             # API 服务
├── assets/          # 静态资源
├── components/      # 通用组件
├── hooks/           # 自定义钩子
├── layouts/         # 布局组件
├── locales/         # 国际化资源
├── pages/           # 页面组件
├── routes/          # 路由配置
├── App.tsx          # 应用入口
├── global-config.ts # 全局配置
└── main.tsx         # 主入口
```

## 核心功能

1. **用户认证**：登录、注册、密码重置
2. **权限管理**：基于角色和权限的访问控制
3. **导航系统**：支持水平、垂直和移动导航
4. **多标签页**：支持多标签页管理
5. **数据可视化**：集成图表组件
6. **文件上传**：支持多种上传方式
7. **国际化**：支持中英文切换
8. **主题管理**：支持明暗主题和主题颜色定制

## 适用场景

- 企业后台管理系统
- 内容管理系统
- 数据管理平台
- 任何需要管理界面的应用
